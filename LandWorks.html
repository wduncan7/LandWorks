📊 Case Intelligence Database — 0 cases loaded

Real rezoning case outcomes from Wake County municipalities. The more cases you add, the more accurate your project evaluations become. Raleigh approves ~96% of cases that reach a vote — you need denied cases from Cary, Wake Forest, and others to build a useful classifier.

📊 About the Raleigh scoring — important context
Your Phase 1 dashboard shows 16% avg approval probability — but 24 of 25 voted cases were approved (96%). The ML classifier isn't miscalibrated about risk factors — it's miscalibrated about the base rate. It scores everything as risky because it has almost no denied cases to learn from, and its probability function starts at 50% rather than Raleigh's actual 96% base. The Project Evaluator here uses Claude's judgment with the base rate context baked in, which gives more realistic probabilities. To improve the ML classifier: get 15–20 denied cases from Cary and Wake Forest — that anchors the low end of the scale. The one Raleigh denial (Z-25-25, R-10→NX-3, 0.09 acres) is nearly identical to approved case Z-02-25 (0.07 acres, same zone change) — proving the decision was about hearing-room politics, not project specs.
